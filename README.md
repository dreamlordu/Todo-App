Minimalist tasarımı ve kullanıcı dostu arayüzü ile günlük görevlerinizi yönetmenizi sağlayan web tabanlı bir Todo uygulaması. **Vite**, **Tailwind CSS** ve **daisyUI** kullanılarak geliştirilmiştir.

## 🚀 Özellikler

- **Görev Ekleme:** Hızlıca yeni görevler oluşturun.
- **Görev Silme:** Tamamlanan veya iptal edilen görevleri tek tıkla kaldırın.
- **Canlı Filtreleme:** Görevleriniz arasında anlık arama yapın.
- **Tümünü Temizle:** Tek butonla listeyi sıfırlayın.
- **Veri Kalıcılığı:** Tarayıcı kapansa bile verileriniz `localStorage` sayesinde güvende kalır.
- **Dinamik Bildirimler:** İşlemlerin sonucuna göre (Başarılı, Uyarı, Hata) şık alert mesajları.

## 🛠️ Kullanılan Teknolojiler

Uygulama modern web teknolojileri ve best-practice yaklaşımları ile inşa edilmiştir:

* **Vite:** Hızlı geliştirme ve build süreci için.
* **Tailwind CSS v4:** Modern ve esnek stil yönetimi.
* **daisyUI:** Hazır ve estetik UI bileşenleri için.
* **Vanilla JavaScript:** Temel mantık ve DOM manipülasyonu.
* **LocalStorage:** Kullanıcı verilerini yerel depolamada saklamak için.
